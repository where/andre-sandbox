
Scripts:

  Basic:
    ping.sh - Pings the REST service
    get.sh - Gets a profile
    put.sh - Adds a new profile - see put-profile.json
    delete.sh - Deletes a profile

  Misc
    get-wadl.sh - Gets the WADL and puts in wadl.xml
    wadl2html.sh - Creates a HTML file wadl.xml.html from wadl.xml

  When you run a curl script, output will be in:
    log.txt - Response body
    log-headers.txt - Response headers

Files
  put-profile.json - Template to PUT a new profile
  sample-get-profile.json - Sample JSON returned by GET
